# 机器学习中的统计学

> 统计学是你判断模型是真的有效还是只是运气好的方式。

**类型：** 实践
**语言：** Python
**前置条件：** 第 1 阶段，第 06 课（概率与分布）、第 07 课（贝叶斯定理）
**时间：** ~120 分钟

## 学习目标

- 从零计算描述统计量、Pearson/Spearman 相关系数和协方差矩阵
- 进行假设检验（t 检验、卡方检验），正确解读 p 值和置信区间
- 使用自举重采样构建任意度量的置信区间，无需分布假设
- 用效应量区分统计显著性和实际显著性

## 问题所在

你训练了两个模型。模型 A 在测试集上得分 0.87，模型 B 得分 0.89。你部署了模型 B。三周后，生产指标比以前更差了。发生了什么？

模型 B 实际上并没有超越模型 A。那 0.02 的差异是噪声。你的测试集太小，或者方差太高，或者两者兼有。你把伪装成改进的随机性给部署上线了。

这种情况不断发生。Kaggle 排行榜大洗牌。论文无法复现。A/B 测试根据几百个样本宣布获胜者。根本原因总是一样的：有人跳过了统计学。

统计学给你提供区分信号和噪声的工具。它告诉你差异何时是真实的，你应该有多大把握，以及在你可以信任结果之前需要多少数据。每个 ML 流水线、每次模型比较、每个实验都需要统计学。没有它，你只是在猜测。

## 核心概念

### 描述统计：汇总你的数据

在建模任何东西之前，你需要了解你的数据是什么样子的。描述统计将数据集压缩为几个捕捉其形状的数字。

**集中趋势的测量**回答"中间在哪里？"

```
均值：  所有值之和 / 计数
        mu = (1/n) * sum(x_i)

中位数：排序后的中间值
        对异常值鲁棒。如果你有 [1, 2, 3, 4, 1000]，均值是 202
        但中位数是 3。

众数：  最频繁出现的值
        对分类数据有用。对连续数据，很少有信息量。
```

均值是平衡点，中位数是中间标志。当它们发散时，你的分布是偏斜的。收入分布均值 >> 中位数（由亿万富翁引起的右偏）。训练期间的损失分布常常均值 << 中位数（由简单样本引起的左偏）。

**离散程度的测量**回答"数据有多分散？"

```
方差：  偏离均值的平均平方差
        sigma^2 = (1/n) * sum((x_i - mu)^2)

标准差：方差的平方根
        sigma = sqrt(sigma^2)
        与数据单位相同，因此更易解读。

极差：  最大值 - 最小值
        对异常值敏感。几乎永远不单独使用。

IQR：   Q3 - Q1（四分位距）
        数据中间 50% 的范围。
        对异常值鲁棒。用于箱线图和异常值检测。
```

**百分位数**将排序数据分成 100 个等份。第 25 百分位数（Q1）意味着 25% 的值低于此点。第 50 百分位数是中位数。第 75 百分位数是 Q3。

```
对于延迟监控：
  P50 = 中位延迟        （典型用户体验）
  P95 = 第 95 百分位数  （糟糕但不是最坏情况）
  P99 = 第 99 百分位数  （尾部延迟，通常是中位数的 10 倍）
```

在 ML 中，你关心推理延迟、预测置信度分布以及理解错误分布的百分位数。平均误差低但 P99 误差糟糕的模型对于安全关键应用可能毫无用处。

**样本 vs 总体统计量。** 从样本计算方差时，除以 (n-1) 而不是 n。这是贝塞尔修正。它补偿了样本均值不是真实总体均值这一事实。分母用 n，你会系统地低估真实方差。用 (n-1)，估计是无偏的。

```
总体方差：sigma^2 = (1/N) * sum((x_i - mu)^2)
样本方差：s^2     = (1/(n-1)) * sum((x_i - x_bar)^2)
```

实践中：如果 n 很大（数千个样本），差异可以忽略不计。如果 n 很小（几十个样本），这很重要。

### 相关性：变量如何协同变化

相关性测量两个变量之间线性关系的强度和方向。

**Pearson 相关系数**测量线性关联：

```
r = sum((x_i - x_bar)(y_i - y_bar)) / (n * s_x * s_y)

r = +1：  完美的正线性关系
r = -1：  完美的负线性关系
r =  0：  没有线性关系（但可能存在非线性关系！）

范围：[-1, 1]
```

Pearson 假设关系是线性的且两个变量大致正态分布。它对异常值敏感。单个极端点可以将 r 从 0.1 拖到 0.9。

**Spearman 秩相关**测量单调关联：

```
1. 用秩（1, 2, 3, ...）替换每个值
2. 对秩计算 Pearson 相关

Spearman 捕捉任何单调关系，不只是线性的。
如果 y = x^3，Pearson 给 r < 1 但 Spearman 给 rho = 1。
```

**何时使用哪种：**

```
Pearson：    两个变量都是连续的且大致正态。
            你特别关心线性关系。
            没有极端异常值。

Spearman：   有序数据（排名、评分）。
            数据不是正态分布的。
            你怀疑是单调但非线性的关系。
            存在异常值。
```

**黄金法则：** 相关性不意味着因果性。冰淇淋销售和溺亡死亡相关，因为两者在夏天都增加。模型准确率与参数数量相关，但增加参数不会自动提高准确率（见：过拟合）。

### 协方差矩阵

两个变量之间的协方差测量它们一起变化的情况：

```
Cov(X, Y) = (1/n) * sum((x_i - x_bar)(y_i - y_bar))

Cov(X, Y) > 0：  X 和 Y 倾向于一起增加
Cov(X, Y) < 0：  当 X 增加时，Y 倾向于减少
Cov(X, Y) = 0：  没有线性协同运动
```

对于 d 个特征，协方差矩阵 C 是一个 d x d 矩阵，其中 C[i][j] = Cov(feature_i, feature_j)。对角线条目 C[i][i] 是每个特征的方差。

```
C = | Var(x1)      Cov(x1,x2)  Cov(x1,x3) |
    | Cov(x2,x1)  Var(x2)      Cov(x2,x3) |
    | Cov(x3,x1)  Cov(x3,x2)  Var(x3)     |

性质：
  - 对称：C[i][j] = C[j][i]
  - 半正定：所有特征值 >= 0
  - 对角线 = 方差
  - 非对角线 = 协方差
```

**与 PCA 的联系。** PCA 对协方差矩阵进行特征分解。特征向量是主成分（最大方差的方向）。特征值告诉你每个成分捕捉了多少方差。这正是第 10 课所涵盖的，但现在你明白了为什么协方差矩阵是正确的分解对象：它编码了数据中所有成对的线性关系。

**与相关性的联系。** 相关矩阵是标准化变量（每个除以其标准差）的协方差矩阵。相关性归一化协方差，使所有值落在 [-1, 1] 内。

### 假设检验

假设检验是在不确定性下做决策的框架。你从一个主张开始，收集数据，然后确定数据是否与该主张一致。

**设置：**

```
零假设 (H0)：     默认假设，通常是"没有效应"
备择假设 (H1)：   你想证明的东西

示例：
  H0：模型 A 和模型 B 具有相同的准确率
  H1：模型 B 的准确率高于模型 A
```

**p 值**是假设 H0 为真时，观察到与你所见数据一样极端的数据的概率。它不是 H0 为真的概率。这是统计学中最常见的单一误解。

```
p 值 = P(数据这么极端 | H0 为真)

如果 p 值 < alpha（通常 0.05）：
    拒绝 H0。结果"统计显著"。
如果 p 值 >= alpha：
    无法拒绝 H0。你没有足够的证据。
    这不意味着 H0 为真。
```

**置信区间**给出参数的合理值范围：

```
均值的 95% 置信区间：
    x_bar +/- z * (s / sqrt(n))

其中 95% 置信度时 z = 1.96

解读：如果你多次重复这个实验，95% 的
计算区间会包含真实均值。这不意味着
真实均值在这个特定区间内的概率是 95%。
```

置信区间的宽度告诉你精度。宽区间意味着高不确定性。窄区间意味着你的估计是精确的（但如果你的数据有偏差，不一定是准确的）。

### t 检验

t 检验比较均值。有几种形式。

**单样本 t 检验：** 总体均值是否与假设值不同？

```
t = (x_bar - mu_0) / (s / sqrt(n))

自由度 = n - 1
```

**双样本 t 检验（独立）：** 两组均值是否不同？

```
t = (x_bar_1 - x_bar_2) / sqrt(s1^2/n1 + s2^2/n2)

这是 Welch t 检验，不假设方差相等。
除非有特定理由假设等方差，否则始终使用 Welch t 检验。
```

**配对 t 检验：** 当测量成对出现时（同一模型在相同数据划分上的评估）：

```
对每对计算 d_i = x_i - y_i
然后对 d_i 值运行 mu_0 = 0 的单样本 t 检验
```

在 ML 中，配对 t 检验很常见：你在相同的 10 个交叉验证折叠上运行两个模型，并成对比较它们的分数。

### 卡方检验

卡方检验检查观测频率是否与期望频率匹配。对分类数据有用。

```
chi^2 = sum((观测 - 期望)^2 / 期望)

示例：语言模型的输出分布是否与
各类别的训练分布匹配？

类别     观测   期望
正面      120    100
负面       80    100
chi^2 = (120-100)^2/100 + (80-100)^2/100 = 4 + 4 = 8

自由度为 1，chi^2 = 8 给出 p < 0.005。
差异显著。
```

### ML 模型的 A/B 测试

ML 中的 A/B 测试与 Web A/B 测试不同。模型比较有特定挑战：

```
1. 相同测试集：  两个模型必须在相同数据上评估。
               不同测试集使比较毫无意义。

2. 多个指标：   仅准确率不够。你需要精确率、
               召回率、F1、延迟和公平性指标。

3. 方差：      使用交叉验证或自举来估计每个指标的
              方差，而不仅仅是点估计。

4. 数据泄露：  如果测试集在模型选择期间被使用，
              你的比较是有偏的。留出最终测试集。
```

**流程：**

```
1. 定义你的指标和显著性水平（alpha = 0.05）
2. 在相同的 k 折交叉验证划分上运行两个模型
3. 收集配对分数：[(a1, b1), (a2, b2), ..., (ak, bk)]
4. 计算差异：d_i = b_i - a_i
5. 对差异运行配对 t 检验
6. 检查：均值差异是否与 0 显著不同？
7. 计算均值差异的置信区间
8. 计算效应量（Cohen's d）来判断实际显著性
```

### 统计显著性 vs 实际显著性

结果可以统计显著但实际上毫无意义。有了足够的数据，即使是微不足道的差异也会变得统计显著。

```
示例：
  模型 A 准确率：0.9234
  模型 B 准确率：0.9237
  n = 1,000,000 个测试样本
  p 值 = 0.001

统计显著？是的。
实际显著？0.03% 的改进不值得
部署新模型的工程成本。
```

**效应量**量化差异的大小，与样本量无关：

```
Cohen's d = (均值_1 - 均值_2) / 合并标准差

d = 0.2：  小效应
d = 0.5：  中等效应
d = 0.8：  大效应
```

始终同时报告 p 值和效应量。p 值告诉你差异是否真实。效应量告诉你差异是否重要。

### 多重比较问题

当你测试许多假设时，一些会"偶然"显著。如果你以 alpha = 0.05 测试 20 件事，即使什么都不真实，你也预期有 1 个假阳性。

```
P(至少一个假阳性) = 1 - (1 - alpha)^m

m = 20 次测试，alpha = 0.05：
P(假阳性) = 1 - 0.95^20 = 0.64

你有 64% 的机会至少得到一个假阳性。
```

**Bonferroni 修正：** 将 alpha 除以测试次数。

```
调整后的 alpha = alpha / m = 0.05 / 20 = 0.0025

只有当 p 值 < 0.0025 时才拒绝 H0。
保守但简单。测试独立时有效。
```

在 ML 中，当你在多个指标上比较模型、测试许多超参数配置或在多个数据集上评估时，这很重要。

### 自举方法

自举通过有放回地重采样你的数据来估计统计量的抽样分布。不需要对基础分布做假设。

**算法：**

```
1. 你有 n 个数据点
2. 有放回地抽取 n 个样本（某些点出现多次，
   某些根本不出现）
3. 在这个自举样本上计算你的统计量
4. 重复 B 次（通常 B = 1000 到 10000）
5. 自举统计量的分布近似
   抽样分布
```

**自举置信区间（百分位法）：**

```
将 B 个自举统计量排序
95% CI = [第 2.5 百分位数, 第 97.5 百分位数]
```

**为什么自举对 ML 重要：**

```
- 测试集准确率是点估计。自举给你
  置信区间。
- 你不能假设指标分布是正态的（特别是
  AUC、F1、前 k 精度）。
- 自举适用于任何统计量：中位数、两均值之比、
  两个模型 AUC 之差。
- 不需要封闭形式的公式。
```

**用于模型比较的自举：**

```
1. 你有模型 A 和模型 B 在相同测试集上的预测
2. 对每次自举迭代：
   a. 有放回地重采样测试集索引
   b. 计算重采样集上的 metric_A 和 metric_B
   c. 存储 diff = metric_B - metric_A
3. 差异的 95% CI：
   [diffs 的第 2.5 百分位数, diffs 的第 97.5 百分位数]
4. 如果 CI 不包含 0，差异显著
```

这比配对 t 检验更鲁棒，因为它不做任何分布假设。

### 参数检验 vs 非参数检验

**参数检验**假设特定分布（通常是正态分布）：

```
t 检验：     假设数据正态分布（或按 CLT n 足够大）
ANOVA：      假设正态性和等方差
Pearson r：  假设双变量正态性
```

**非参数检验**不做分布假设：

```
Mann-Whitney U：      比较两组（替代独立 t 检验）
Wilcoxon 符号秩：     比较配对数据（替代配对 t 检验）
Spearman rho：        对秩的相关（替代 Pearson）
Kruskal-Wallis：      比较多组（替代 ANOVA）
```

**何时使用非参数检验：**

```
- 样本量小（n < 30）且数据明显非正态
- 有序数据（评分、排名）
- 无法移除的重度异常值
- 偏斜分布
```

**何时使用参数检验：**

```
- 样本量大（CLT 使检验统计量近似正态）
- 数据大致对称，没有极端异常值
- 更强的统计功效（更好地检测真实差异）
```

在 ML 实验中，你通常有小 n（5 或 10 个交叉验证折叠），所以像 Wilcoxon 符号秩这样的非参数检验通常比 t 检验更合适。

### 中心极限定理：实际意义

CLT 说，随着 n 的增长，无论基础总体分布如何，样本均值的分布都趋近于正态分布。

```
如果 X_1, X_2, ..., X_n 是均值为 mu、方差为 sigma^2 的独立同分布：

    X_bar ~ Normal(mu, sigma^2 / n)    当 n -> 无穷

在大多数情况下 n >= 30 时有效。
对于高度偏斜的分布，你可能需要 n >= 100。
```

**为什么这对 ML 重要：**

```
1. 为聚合指标上的置信区间和 t 检验提供依据
2. 解释为什么即使单个折叠变化很大，
   对交叉验证折叠取平均也能给出稳定估计
3. 小批量梯度下降有效是因为批次上的平均梯度
   近似于真实梯度（CLT 的作用）
4. 集成方法：对许多模型的预测取平均
   比任何单个模型给出更稳定的输出
```

**CLT 不做的事：**

```
- 不使你的数据正态。它使样本均值正态。
- 不适用于方差无限的重尾分布
  （柯西分布）。
- 不适用于相关数据（未经修正的时间序列）。
```

### ML 论文中常见的统计错误

1. **在训练集上测试。** 保证过拟合。始终留出模型在训练期间从未见过的数据。

2. **没有置信区间。** 报告没有不确定性的单一准确率数字使结果无法复现和验证。

3. **忽略多重比较。** 测试 50 种配置并报告最好的一个，不加修正会夸大假阳性率。

4. **混淆统计显著性和实际显著性。** 0.01% 准确率提升的 p 值 0.001 没有意义。

5. **在不平衡数据上使用准确率。** 在有 99% 负类的数据集上达到 99% 准确率意味着模型什么都没学到。使用精确率、召回率、F1 或 AUC。

6. **挑选指标。** 只报告模型获胜的指标。诚实的评估报告所有相关指标。

7. **跨训练/测试集泄露信息。** 在划分前归一化，或使用未来数据预测过去。

8. **没有方差估计的小测试集。** 在 100 个样本上评估并声称 2% 的改进是噪声，而不是信号。

9. **假设数据独立而实际不独立。** 来自同一患者的医学图像，来自同一文档的多个句子。同一组内的观测是相关的。

10. **P 值操控。** 尝试不同的检验、子集或排除标准，直到得到 p < 0.05。结果是搜索的产物。

你将实现：

1. **从零实现描述统计**（均值、中位数、众数、标准差、百分位数、IQR）

| 均值 | 值的和除以计数。对异常值敏感。 |
| 中位数 | 排序数据的中间值。对异常值鲁棒。 |
| 标准差 | 方差的平方根。以原始单位测量离散程度。 |
| 百分位数 | 给定百分比的数据落在此值以下。 |
| IQR | 四分位距。Q3 减 Q1。中间 50% 的数据范围。 |
| Pearson 相关 | 测量两个变量之间的线性关联。范围 [-1, 1]。 |
| Spearman 相关 | 使用秩测量单调关联。 |
| 协方差矩阵 | 所有特征之间成对协方差的矩阵。 |
| 零假设 | 没有效应或没有差异的默认假设。 |
| p 值 | 给定零假设为真时，数据这么极端的概率。 |
| 置信区间 | 在给定置信水平下参数的合理值范围。 |
| t 检验 | 检验均值是否显著不同。使用 t 分布。 |
| 卡方检验 | 检验观测频率是否与期望频率不同。 |
| 效应量 | 差异的大小，与样本量无关。Cohen's d 很常用。 |
| Bonferroni 修正 | 将显著性阈值除以测试次数以控制假阳性。 |
| 自举 | 有放回重采样以估计抽样分布。 |
| I 类错误 | 假阳性。当 H0 为真时拒绝它。 |
| II 类错误 | 假阴性。当 H0 为假时无法拒绝它。 |
| 统计功效 | 正确拒绝假 H0 的概率。功效 = 1 减 II 类错误率。 |
| 中心极限定理 | 随着样本量增长，样本均值收敛到正态分布。 |
| 参数检验 | 假设数据的特定分布（通常是正态分布）。 |
| 非参数检验 | 不做分布假设。适用于秩或符号。 |
